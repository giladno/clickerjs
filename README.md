# clickerjs
Remote debugging and editing for Node.js projects

![clientjs](https://raw.githubusercontent.com/giladno/clickerjs/master/screenshots/screen1.jpg?raw=true "clickerjs")

## Installing

### Prerequisites
- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/) (preferably via [nvm](https://github.com/creationix/nvm) or [nave](https://github.com/isaacs/nave))
- [npm](http://npm.im/)
- [Bower](http://bower.io/)

### Instructions
```
git clone git@github.com:giladno/clickerjs.git
npm install && bower install
cd clickerjs
./clicker.js ~
```
Then navigate your browser to http://YOUR_SERVER_IP:8085/
